# Упрощенный таск-трекер "Задачник"
## Описание проекта
Настолькое Python-приложение для ведения задач, проектов и исполнителей с возможностью просмотра и визуализацией данных.

## Возможности
- Добавление задач
- Добавление проектов
- Добавление испольнителей
- Назначение задач исполнителям
- Измнение статусов задач
- Анализ прогресса выполнения проектов
- GUI Tk

## Запуск
```
pip install -r requirements.txt
python main.py
```
